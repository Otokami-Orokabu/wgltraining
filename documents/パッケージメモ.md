# パッケージメモ

使用しているパッケージについてのメモ

## node

- 初期化
    ``` shell
    npm init -y
    ```

- typescript 定義
    ``` shell
        npm i -D ts-node
    ```
    ※ module: "CommonJS" でないとimport問題が解決できんらしい

## typescript

``` shell
    npm i -D typescript
```

## jest

``` shell
    npm i -D jest @types/jest ts-jest
```
## webpack

``` shell
    npm i -D webpack ts-loader @types/webpack
```
